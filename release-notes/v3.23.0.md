# AGET v3.23.0 — Goal Tier (preview)

**Theme**: Goal Tier (preview)
**Type**: Minor (no breaking changes)
**Released**: 2026-06-20

## What's New

- **Goal Tier (preview)** — a new `aget-create-goal` skill commits a selected candidate goal into a governed Goal artifact (North Star → Goal → Initiative), backed by `AGET_GOAL_SPEC` v0.2.0 (draft). The canonical spec promotion follows in 3.23.1.
- **`aget-propose-goals` skill** — a plural candidate-goal generator: proposes N ranked session-goal candidates, each scored ex-ante by a goal-selection rubric.

## Compatibility

- Minor release, **no breaking changes** — upgrade from 3.22.0 is drop-in.
- The Goal Tier ships as a **preview**: the two skills land in canonical `aget/` core this release; canonical `AGET_GOAL_SPEC` promotion and template skill propagation land in 3.23.1.

## Notes

- Reduced scope this cycle: the self-oversight ratchet generalization and the interaction-channel rung are deferred to 3.24.
- A close-time authorization guard was built and exercised in the framework-manager instance this cycle; it is **not** part of this public release — it lands in canonical (with a population-spanning pilot) in 3.23.1.
