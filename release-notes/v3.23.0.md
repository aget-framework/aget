### C-23-PG — `/aget-propose-goals` skill (SP-017) — *attached to v3.23 (principal directive 2026-06-15)*

The goal-altitude sibling of `/aget-propose-actions`: a **plural candidate-set generator** that produces N ranked candidate session-goals, scores each ex-ante with `RUBRIC_goal_selection_v1.0` (5 dimensions, /15), and records the scored set in the session file for retrospective audit.

- **What it is**: divergent-proposal mode (L677) at goal altitude — "which goal should this session pursue?" Default count = 3.
- **Plurality (L1067)**: PLURAL candidate-set — no `create` sibling of its own. A selected goal funnels into the downstream `create-goal` committer when the Goal-tier (INIT-CORE-ARTIFACT-MATURATION Stream 9) lands. Built **standalone** (principal D1, 2026-06-15) — its only hard dependency, the rubric, already existed.
- **Constraints enforced**: C1 rubric-mandatory (unscored goal = decorative, L671) · C2 session-recording mandatory · C3 N≥2 (else `/aget-propose-project`) · C4 N≤7 (Miller 1956) · C5 explicit tie-break · no execute-all default (goals are selected, not batch-executed).
- **Governance enhancements (E1/E2/E4/E7/E8/E9, built 2026-06-17)** — raises the skill from value-ranking to goal-governance by folding in the sibling `SOP_near_term_ambition_projection`'s three gates (L1090, one intent at two altitudes):
  - **E7 commitment-type honesty** — each candidate tagged `committed`|`aspirational` (orthogonal to the /15 score); an aspirational goal may not be presented as committed (OKR honesty, C6/PG-011).
  - **E8 unshaped→shape-it gate** — an aspirational/ungoverned recommended target routes to a shape-it action (`/aget-propose-project`), never direct execution (L174 ambition–governance inversion, C7/PG-012).
  - **E1 structural recording** — `scripts/record_goals_ext.py` makes C2 a checkable surface with a negative control that bites (L605/L671 close).
  - **E9 appetite-flexes-scope** (Shape Up) on the budget parameter; **E4** ontology grounding (`aget:concept/Principal`). E2 auto-fire trigger documented (advisory; true hook deferred to self_oversight infra, L474–476).
- **Verification**: `tests/test_propose_goals.py` — **27 firing falsifiers PASS** (14 v1.0 + 13 governance); dogfooded reflexively; ADR-007 negative controls verified per gate.
- **Artifacts**: `.claude/skills/aget-propose-goals/SKILL.md`; `.aget/specs/skills/SKILL-055_aget-propose-goals.yaml`; `scripts/record_goals_ext.py`; `rubrics/RUBRIC_goal_selection_v1.0.md`; SP-017 → Complete.
