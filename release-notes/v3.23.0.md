# AGET v3.23.0 — Goal Tier & Integrity Hardening

**Theme**: Goal Tier & Integrity Hardening
**Type**: Minor (no breaking changes)
**Released**: 2026-06-20

## What's New

- **Goal Tier (preview)** — a new `aget-create-goal` skill commits a selected candidate goal into a governed Goal artifact (North Star → Goal → Initiative), backed by `AGET_GOAL_SPEC` v0.2.0 (draft). The canonical spec promotion follows in 3.23.1.
- **`aget-propose-goals` skill** — a plural candidate-goal generator: proposes N ranked session-goal candidates, each scored ex-ante by a goal-selection rubric.
- **Close-time authorization guard** — `/aget-close-project` now requires a linked authorization event for principal-attributed closes and flags irreversible consequences, strengthening release-close integrity.

## Fixed

- Scope-board generation against a locked version scope (#1654).
- Homepage `--apply` no-op on multi-section content updates (#1656).

## Compatibility

- Minor release, **no breaking changes** — upgrade from 3.22.0 is drop-in.
- The Goal Tier ships as a preview (skill + draft spec); canonical `AGET_GOAL_SPEC` promotion and template skill propagation land in 3.23.1.
